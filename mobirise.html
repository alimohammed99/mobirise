HOW MOBIRISE WORKS?



Drag blocks to page

Start with creating a new website and picking
up the theme. Then expand the blocks panel
with the big red "plus" button in the lower right
corner and start dragging the blocks you like.






Edit and style inline

Edit the content of each block just you
would in a regular text editor, click on media
element to insert your own image, video or
icon.







Set block parameters

Set the appropriate options through the
parameters panel beingbrought out by the blue
"gear" button.






Preview and Publish

Click on "Mobile View" in the website buiulder to
check how your site looks on different devices
and publish to local drive, FTP or Mobirise
subdomain.
